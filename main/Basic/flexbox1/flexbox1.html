Mở Đầu

Trước đây khi chúng ta layout Website, chia cột hay tạo menu đa cấp phải sử dụng float rất thường xuyên và mình phải nói nó cực kỳ bất tiện. Chưa kể là khi chúng ta làm Responsive sẽ khá mất thời gian để xử lý những đoạn code "dư thừa".

Để giải quyết vấn đề đó thì Flexbox được sinh ra giúp việc layout đơn giản hơn rất nhiều.

Không dài dòng nữa chúng ta cùng tìm hiểu Flexbox làm được những gì nào!

Thuộc tính Flexbox

Trong CSS thuộc tính Flexbox được khai báo như sau display: flex. Mình sẽ demo cho các bạn xem luôn.

Chúng ta có đoạn HTML như sau:

Và CSS như thế này:

.container {
    display: flex;
    width: 800px;
    height: 150px;
    background: black;
}
.item {
    width: 150px;
    height: 150px;
}
.item1 {
    background: coral;
}
.item2 {
    background: lightblue;
}
.item3 {
    background: red;
}
.item4 {
    background: brown;
}

Kết quả như sau:

Sau đây để thì mình xin giới thiệu vài thuộc tính cơ bản của Flexbox để các bạn dễ làm quen nha!

Flex-direction

Đúng như tên của nó là điều hướng flexbox. Flexbox được chia làm hai trục X và Y nhưng nó không giống trong toán học đâu nhé. Trục X là trục ngang đi từ trái qua phải, còn Trục Y thì từ trên xuống dưới nha!

Đây là các giá trị của Flex-direction:

  • row canh theo trục X từ trái qua phải. Đây là giá trị mặc định của Flex-direction.

    • 1
    2
    3
    4

  • row-reverse tương tự như row nhưng theo hướng ngược lại.

    • 1
    2
    3
    4

  • column canh theo trục Y từ trên xuống dưới

    • 1
    2
    3
    4

  • column-reverse tương tự như column nhưng theo hướng ngược lại.

    • 1
    2
    3
    4

Khá là dễ hiểu phải không? Chúng ta cùng qua thuộc tính kế tiếp của Flexbox nào!

Flex-wrap

Thuộc tính này giúp các items xuống hàng khi kích thước của container thay đổi.

NOTE: resize trình duyệt để thấy sự khác biệt nha các bạn.

Flex-flow

Flex-flow thì khá đơn giản thôi, nó là thuộc tính gộp của Flex-direction và Flex-wrap. flex-flow: flex-direction flex-wrap các bạn chỉ cần điền giá trị tương ứng thôi nhé!
