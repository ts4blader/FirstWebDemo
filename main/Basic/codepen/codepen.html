Mở Đầu

Xin chào cả nhà, nếu các bạn đã ở đây thì chắc hẳn ai cũng biết cơ bản về HTML và CSS rồi nhỉ? Nhưng các bạn đã biết cách làm thế nào để website của mình nhìn thêm bát mắt và phong phú chưa? Đúng chúng ta cần một vài Animation nho nhỏ đó.

Animation cơ bản

Thật ra Animation chỉ là 1 phần của JS vì vậy khi mới tiếp cận với Animation thì sẽ khá khó khăn, may thay CSS3 đã hỗ trợ về Animation khá tốt để chúng ta tiếp cận một cách dễ dàng hơn.

Để tiếp cận Animation dễ dàng hơn thì mình xin giới thiệu một vài Tools nhé!

Codepen

Codepen là một trang code web trực tuyến nha các bạn. Hình minh họa codepen:

Các bạn nhìn là hiểu ngay chúng ta có 3 ô code đó là HTML, CSS và JS và một ô Preview là ô chiếm diện tích lớn nhất trên màn hình cũng chính là phần mình thích nhất ở codepen. Các bạn chỉ cần code ở 3 ô kia thì sản phẩm của mình sẽ hiển thị ngay bên ô Preview.

Hơn nữa codepen còn hỗ trợ Emmet và các ngôn ngữ tiền xử lý khá tốt. Emmet là gì mình sẽ nói sau.

Theo mình Codepen là công cụ hỗ trợ code web mạnh nhất nếu bạn học trực tuyến.

VS Code

Khác với Codepen, VS Code là một phần mềm Text Editor nó có khả năng code rất nhiều ngôn ngữ không riêng gì code Web

Với sự hỗ trợ mạnh mẽ từ khả năng Autocomplete và Emmet các bạn sẽ thấy VS Code hỗ trợ bạn code nhanh như thế nào

VS Code có thể code cả một Project mà bạn khó có thể làm thế trên Codepen. Hơn nữa VS Code còn có các Extensions rất thú vị:

  • Live Server
  • Auto Rename Tag
  • CSS Peek
  • Prettier
  • Emmet

Emmet

Chắc các bạn đang thắc mắc Emmet là gì mà được hỗ trợ nhiều như vậy sau đây mình sẽ cho các bạn biết Emmet là gì nha.

Ai cũng thấy việc đóng mở thẻ trong HTML rất là mất thời gian Emmet sinh ra để giải quyết việc đó.

Ví dụ như bạn cần một đoạn HTML như sau:

Ta chỉ cần gõ div.container>div.item*3 sau đó bấm phím Tab là có ngay. Rất tiện phải không nào? Bây giờ không phải lo mở hay đóng thẻ nữa rồi. Có Emmet sẽ cải thiện hiệu suất code của bạn lên rất nhiều đấy!

Bạn cần một thẻ thì cứ việc gõ tên thẻ sau đó gõ phím Tab thì bạn sẽ được một thẻ HTML hoàn chỉnh. Nếu muốn thêm luôn Class thì gõ theo cú pháp tên thẻ.tên class rồi Tab, còn ID thì chỉ cần đổi thành dấu # là được nha!

Các bạn muốn tham khảo thêm về Emmet
