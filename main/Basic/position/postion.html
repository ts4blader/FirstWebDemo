Mở đầu

Chắc hẳn khi các bạn layout trang web hay tạo một menu thì việc sử dụng thuộc tính position hẳn đã không còn xa lạ gì với chúng ta.

Hôm nay chúng ta cùng tìm hiểu thuộc tính Position hoạt động như thế nào nha.

Giá trị của Position

Thuộc tính này chỉ có vài giá trị cũng khá là dễ nhớ:

  • relative: Giá trị này thiết lập vị trí của phần tử mà không gây ảnh hưởng đến các phần tử khác.

  • absolute: Giá trị này thiết lập vị trí của phần tử dựa vào phần tử cha của nó nếu cha của nó đã định position relative hoặc absolute.

  • fixed: Giá trị này thiết lập vị trí của phần tử cố định trên màn hình dù bạn có scroll trình duyệt.

  • static: Đây là giá trị mặc định của position.

Thuộc tính đi chung với Position

Những thuộc tính đi chung với Position:

  • top: Căn chỉnh tử phần tử từ trên xuống dưới nếu giá trị > 0 còn < 0 thì phần tử sẽ đi ngược lại.

  • bottom: Căn chỉnh tử phần tử từ dưới lên trên nếu giá trị > 0 còn < 0 thì phần tử sẽ đi ngược lại.

  • left: Căn chỉnh tử phần tử từ trái sang phải nếu giá trị > 0 còn < 0 thì phần tử sẽ đi ngược lại.

  • right: Căn chỉnh tử phần tử từ phải qua trái nếu giá trị > 0 còn < 0 thì phần tử sẽ đi ngược lại.

    Một vài lưu ý khi sử dụng các thuộc tính đi kèm:

  • Khi cả left và right bằng 0, đã có height mà thuộc tính width chưa định thì phần tử sẽ có width là 100%.

  • Tương tự bottom và top bằng 0, đã có width mà thuộc tính height chưa định thì phần tử sẽ có height là 100%.

Khá là khó hiểu nhỉ. Đừng lo, coi xong ví dụ các bạn sẽ hiểu ngay. Mình chỉ giải thích hai thuộc tính absolute và relative, còn fixed thì khá đơn giản các bạn có thể tự tìm hiểu.

Absolute

Đầu tiên, mình sẽ giải thích bốn thuộc tính top, bottom, left, right.

top: 0

left: 0

top: 0

right: 0

bottom: 0

left: 0

bottom: 0

right: 0

  • top: 0, left: 0 Phần tử sẽ nằm ở bên trên trái cùng.

  • top: 0, right: 0 Phần tử sẽ nằm ở bên trên phải cùng.

  • bottom: 0, left: 0 Phần tử sẽ nằm ở bên dưới trái cùng.

  • bottom: 0, right: 0 Phần tử sẽ nằm ở bên dưới phải cùng.

Các bạn xem tiếp ví dụ này nhé.

Chúng ta có một đoạn HTML sau

Và Style như sau 

.container {
  width : 500px;  
  height : 400px;
  background: black;
}
.item {
  position: absolute;
  width : 100px;  
  height : 100px;
  background: red;
}

Tada!! Ta sẽ được kết quả như sau:

Sau đó chúng ta thử thêm bottom: 0 và left: 0 vào để xem kết quả ra sao nhé.

Các bạn có thấy nó sai cái gì không? Đúng ra nó phải nằm ở góc dưới bên trái của .container chứ nhỉ, sao nó lại nằm ở ngoài body.

Chúng ta không sai mà là thiếu không định position cho phần tử cha của .item. Chỉ cần thêm position: relative là ok thôi.

Relative

origin
top: 100px
left: 100px
right: 100px
bottom: 100px

  • origin: Đây là phần tử ban đầu các giá trị của top, bottom, left, right đều mang giá trị 0.

  • top: 100px Phần tử sẽ cách phần tử origin 100px về phía bên trên.

  • bottom: 100px Phần tử sẽ cách phần tử origin 100px về phía bên dưới.

  • left: 100px Phần tử sẽ cách phần tử origin 100px về phía bên trái.

  • right: 100px Phần tử sẽ cách phần tử origin 100px về phía bên phải.

Note: nếu chỉ định position: relative thì phần tử đó sẽ đứng yên tại chỗ.

Kết

Bài học về position đến đây là hết rồi, tuy những kiến thức này chỉ là cơ bản nhưng nó giúp ích cho việc layout Website rất nhiều nên các bạn cố gắng nắm vững nhé.
